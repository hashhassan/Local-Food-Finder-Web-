What is Local Food Finder?

Local Food Finder is a simple website that helps people discover popular food spots, restaurants, and street food in Rawalpindi and Islamabad. The goal of this project is to make it easier for residents and visitors to find good local food options without having to search through many different apps or websites.

The website covers a wide range of food types including traditional desi food, BBQ, street food, and sweets. All listings are focused specifically on the twin cities of Rawalpindi and Islamabad, so users can find relevant and local information.

About the Project

This website was made as a semester project for a web development course. It is built using only basic HTML and CSS, with no JavaScript or frameworks used. The purpose was to practise building a multi-page website with a shared layout and consistent styling.

The design is kept simple and readable. All food names, locations, and areas mentioned in this project are real places from Rawalpindi and Islamabad. Placeholder images are used since this is a student project.
